# 快速学习

* *强制* 命名单词必须全字母小写，单词以 `-` 分隔。
* *强制* 层次或者父子关系，用 `.` 分隔。
* *建议* 每行不得超过 `140` 个字符。
* *强制* 注意换行。
* *强制* 注释必须独占一行。
* *强制* Html、Css 属性值采用双引号 `""`。
* *强制* JavaScript、TypeScript 字符串采用单引号 `''`。
* *强制* `{` 前后必须要有空格。
* *强制* `(` 前后必须要有空格。
* *强制* `:` 前不允许空格，后必须要有空格。
* *强制* `,` 和 `;` 前不允许有空格。如果不位于行尾，`,` 和 `;` 后必须要有空格。
